# Systematic Literature Review (SLR) – Paper Collection Repository

This repository contains curated reference papers used in multiple **Systematic Literature Review (SLR)** studies focusing on **Applied AI**. Each folder represents an independent SLR topic with its own scope, paper selection, and references.

---

## 📁 Repository Structure

Each directory corresponds to a **specific SLR topic**:
Inside each folder, you may find:
- `README.md` — Topic-specific description and scope  
- `references.bib` — BibTeX references used in the SLR
